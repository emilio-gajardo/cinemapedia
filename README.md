# cinededomena

### Info
* Aplicación móvil creada con Flutter y Dart

## Dev
1. Copiar el .env.template y renombrar a .env
2. Cambiar las variables de entorno (The MovieDB)
3. Ejecutar:
```
flutter pub run build_runner build
```

## Instalaciones de plugins en vs code
* REST client
* Dotenv Official
* Dart
* Flutter
